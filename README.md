# DigitalResourcesTest

## Run frontend

```bash
cd frontend
yarn
yarn run dev
```

## Run Backend

```bash
cd backend
yarn
yarn run start
```

#### Note

_The default Mongo Atlas Uri can be changed by setting the following environment variable_
`MONGO_URI`
